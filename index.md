# Markdown Tutorial

## Headers
* You can add headers by typing `#` 1-6 times followed by a space
  
### This is the third largest header `<h3>`
###### This is the smallest header `<h6>`

---

## Images
* To add an image, type `![Description](url)`. Make sure to inlcude the `!` beforehand.
